# SentinelAI – AI-Powered Security Operations Center

A cloud-native security platform that ingests system and network logs in real time, detects malicious activity using machine learning, explains threats with an LLM, and provides analysts with an interactive dashboard for investigation.

---

## Tech Stack

### Frontend
* **Framework:** React + Vite
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **State Management:** React Query
* **Data Visualization:** Chart.js / Recharts

### Backend & AI Service
* **Core API:** Node.js, Express, TypeScript
* **AI & ML Engine:** Python, FastAPI
* **Machine Learning:** PyTorch / Scikit-Learn
* **LLM Integration:** OpenAI API *(with plans for local LLM deployment)*

### Database & Analytics
* **Primary Database:** PostgreSQL
* **Caching Layer:** Redis
* **Search & Log Analytics:** Elasticsearch
